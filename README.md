# Kişisel Dizi İzleme ve Analiz Paneli

## Problem Tanımı
Bireysel izleyicilerin takip ettikleri dizileri, harcadıkları zamanı ve verdikleri puanları manuel olarak takip etmesinin zor olması.

## Çözüm ve Ürün
Kullanıcıların kendi izleme verilerini (tür, bölüm sayısı, puan) yükleyerek istatistiklerini görebildiği ve bu verilere dayanarak yapay zekadan yeni dizi önerileri alabildiği bir web paneli.

## Kullanılan Teknolojiler
* Python
* Pandas (Veri analizi)
* Streamlit (Web arayüzü)
* Matplotlib (Görselleştirme)
* LLM API (Öneri sistemi)
